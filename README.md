# project\_V5\_15

Project for Module V5\_15 "Developing Software as s Product"



## Description



This game is called “Game of Life.” Various structures from the real world are replicated on a playing field, such as land, water, houses, roads, people, and cars. The game can be started either with a real map or a random map. Every second, the playing field is updated and displayed in the terminal. With each update of the map, it is changed by moving vehicles, a growing population, a dynamic economy, etc. The game is currently still quite simple and has some features that could be improved. The graphics of the game are also not yet fully developed.

1. Clone the repository:
```
git clone https://github.com/anika-dot/project_V5_15.git
cd project_V5_15
```

3. Create a virtual environment
```

source venv/bin/activate   # Mac/Linux
```
